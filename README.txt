# ProFront Performance Audit Extension
Bu eklenti, front-end geliştiriciler için yerel performans ve kod analizi yapar.

## Kurulum
1. Chrome'da chrome://extensions/ adresini açın.
2. Sağ üstten 'Geliştirici Modu'nu açın.
3. 'Paketlenmemiş öğe yükle' diyerek bu klasörü seçin.

## Özellikler
- Sayfa yükleme hızını milisaniye cinsinden ölçer.
- DOM ağacı büyüklüğünü denetler.
- Güvenlik ve performans hatalarını (console.log, büyük dosyalar vb.) raporlar.
